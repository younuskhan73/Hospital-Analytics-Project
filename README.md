# Hospital Analytics Project



#### Hospital Operations \& Revenue Analytics Dashboard using Power BI





&nbsp;Project Overview



This project focuses on analysing hospital operations, patient admissions, and revenue performance using Power BI.  

The dashboard provides executive-level insights to support data-driven decision-making.

\- Track revenue trends (MoM \& YoY)
